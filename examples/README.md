# DevSecOps Dev Kit - Template terraform module example

This example creates 3 null resources.

## Usage

To run this example you need to execute:

```bash
$ terraform init
$ terraform plan
$ terraform apply
```

After testing with this example is completed, run ```$ terraform destroy```
